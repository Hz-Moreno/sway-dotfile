### FEDORA - SWAY
Minimalist, performance-driven configurations for Sway WM on Fedora Linux, optimized for a Fullstack development workflow.

### Core Stack
**Window Manager**: Sway (Wayland)

**Terminal Emulator**: foot

**Application Launcher**: rofi-wayland

**Lockscreen**: hyprlock

**Idle Daemon**: swayidle (with auto-kill logic for safe reloading)

**Login Launcher**: SDDM

### Dependencies

* pywal 
* hyperlock
* awww
* foot
* fzf
* zsh
* rofi-wayland
* git
* wl-clipboard
* swayidle
* SDDM
* waybar
* fzf

### Installation

Clone this repo:

```bash
git clone https://github.com/Hz-Moreno/sway-dotfile.git
```

And run:

```bash
./install.sh
```
